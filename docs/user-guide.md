# ENTP 英語学習アプリ - ユーザーガイド

## 🎯 アプリの概要

ENTP 英語学習アプリは、ENTP（外向的直感思考知覚）タイプの学習者が飽きずに楽しく英語学習を継続できる Progressive Web App (PWA)です。

### 🌟 主な特徴

- **多様な学習モード**: 語彙、文法、リスニング、英作文、TOEIC 模擬テスト
- **ゲーミフィケーション**: XP、アチーブメント、ガチャ、ハートシステム
- **統合システム**: 学習モード間のシナジー効果と AI 推薦
- **PWA 対応**: インストール可能、オフライン学習対応

## 🚀 始め方

### 1. アプリのインストール

1. ブラウザでアプリを開く
2. アドレスバーの「インストール」ボタンをクリック
3. デスクトップまたはモバイルにアプリをインストール

### 2. 初回設定

- **レベル**: 現在の英語レベルを選択
- **目標**: 学習目標を設定（TOEIC、日常会話など）
- **学習時間**: 1 日の学習時間を設定

## 📚 学習機能の使い方

### 🎯 語彙学習

1. **ホーム画面**から「語彙学習」を選択
2. **難易度**を選択（初級・中級・上級）
3. **カテゴリー**を選択（ビジネス、日常、学術など）
4. **学習モード**を選択：
   - **基本モード**: 従来の語彙学習
   - **ガチャモード**: カード収集要素付き

### 📖 文法クイズ

1. **文法クイズ**を選択
2. **カテゴリー**を選択（基本文型、時制、助動詞など）
3. **文型**を選択（該当する場合）
4. **難易度**を選択
5. **問題集**を選択してクイズ開始

### 🎧 リスニング学習

1. **リスニング学習**を選択
2. **難易度**を選択
3. **パート**を選択（Part 1-4）
4. 音声を聞いて問題に解答

### ✍️ 英作文

1. **英作文**を選択
2. **トピック**を選択
3. 制限時間内で作文を完成
4. **AI 添削**機能でフィードバック受信

### 📊 TOEIC 模擬テスト

1. **TOEIC 模擬テスト**を選択
2. **テスト設定**：
   - テストタイプ（リスニング・リーディング・フル）
   - 問題数と時間制限
   - 難易度設定
3. **事前学習連携**を確認（ブースト効果）
4. **テスト開始**

### 🎯 学習パスチャレンジ

1. **学習パスチャレンジ**を選択
2. **推奨チャレンジ**を確認
3. **前提条件**を満たす学習を完了
4. **チャレンジ開始**

## 🎮 ゲーミフィケーション要素

### ⭐ XP システム

- **獲得方法**: 学習完了、正解、継続学習
- **用途**: レベルアップ、ショップでの購入
- **レベル効果**: 新しい機能の解放

### 🏆 アチーブメント

- **種類**: 学習継続、正解率、特定分野の習得
- **報酬**: XP、特別なアイテム、称号
- **確認方法**: 「実績」画面で進捗確認

### 🎴 ガチャシステム

- **コイン**: 学習完了で獲得
- **パック**: コイン消費で語彙カードを入手
- **レアリティ**: 通常・レア・超レア・伝説
- **活用**: 収集したカードで学習ブースト

### ❤️ ハートシステム

- **回復**: 時間経過またはアイテム使用
- **消費**: 学習モードの利用で消費
- **戦略**: 効率的な学習計画を立てる

### 🎯 デイリーチャレンジ

- **更新**: 毎日新しいチャレンジ
- **種類**: 語彙学習、文法クイズ、リスニングなど
- **報酬**: コイン、XP、特別なアイテム

## 🔗 統合システム

### ✨ シナジー効果

- **事前学習 ↔ TOEIC**: 理論学習で TOEIC 問題の精度向上
- **スキルツリー ↔ 文法**: スキル習得で文法問題の難易度軽減
- **ガチャ ↔ リスニング**: 語彙カード収集でリスニング理解度向上
- **デイリーチャレンジ ↔ 全モード**: 継続学習で全体的なブースト

### 🤖 インテリジェント推薦

- **弱点分析**: 学習履歴から苦手分野を特定
- **問題推薦**: 個人に最適化された問題を提案
- **学習パス**: 効率的な学習順序を推奨

### 📈 統合ダッシュボード

- **進捗確認**: 全学習モードの進捗を一覧表示
- **シナジー効果**: アクティブなブースト効果を確認
- **統計情報**: 学習時間、正解率、継続日数など

## 📱 PWA 機能

### 💾 オフライン学習

- **事前ダウンロード**: 学習コンテンツをローカルに保存
- **オフライン利用**: インターネット接続なしでも学習可能
- **同期**: オンライン復帰時に進捗を同期

### 🔔 通知機能

- **学習リマインダー**: 設定した時間に学習を促進
- **アチーブメント通知**: 達成時に通知表示
- **デイリーチャレンジ**: 新しいチャレンジの通知

## 🎯 学習戦略のコツ

### 📅 継続のコツ

1. **デイリーチャレンジ**を毎日完了
2. **ハートシステム**を活用して効率的に学習
3. **アチーブメント**を意識して目標設定

### 🚀 効率化のコツ

1. **事前学習**で理論を固めてから実践
2. **シナジー効果**を活用して複数モードを連携
3. **インテリジェント推薦**で弱点を集中的に学習

### 🎮 楽しさのコツ

1. **ガチャシステム**で語彙カードを収集
2. **タワーディフェンス**で学習要素を含むゲーム
3. **レベルアップ**で新しい機能を解放

## 🆘 トラブルシューティング

### 🔧 よくある問題

**Q: アプリが重い・遅い**
A: ブラウザのキャッシュをクリアしてください。PWA の場合、アプリを再インストールしてみてください。

**Q: 学習進捗が消えた**
A: ローカルストレージの容量不足の可能性があります。ブラウザのストレージを確認してください。

**Q: 音声が再生されない**
A: ブラウザの音声許可設定を確認してください。また、デバイスの音量設定も確認してください。

**Q: オフラインで使えない**
A: 事前にオンラインでアプリを開いて、Service Worker が登録されていることを確認してください。
