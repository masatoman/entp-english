# ENTP英語学習アプリ

ENTPタイプの学習者が**3ヶ月継続できる**超シンプルな英語学習PWAアプリ。

**コンセプト**: 1日9分（3分×3回）で英語力向上

## 🚀 クイックスタート

```bash
npm install
npm run dev
```

## 🎮 機能（超シンプル版）

- **3分クイズ** - 文法・単語のミックス
- **ガチャシステム** - 単語収集のゲーム性
- **成長グラフ** - 予測TOEICスコア
- **今日の目標** - 朝・昼・夜の3回
- **ストリーク表示** - 継続日数
